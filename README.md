PIXEL PHOTOBOOTH

Welcome to the **Pixel Photobooth**! This is a web-based, fully interactive digital photobooth designed with a cozy, retro 8-bit arcade aesthetic. It features customizable frame overlays, ambient lighting controls, and aesthetic filters.

This app runs entirely in your browser (**client-side**), meaning your photos are processed locally on your device and are never uploaded to a server. Your privacy is 100% secure!

Features

*   **Retro UI:** Built with custom pixel styles using a dedicated palette (`#770737`, `#F8C8DC`, `#673147`).
*   **Aesthetic Filters:** Choose between 8-Bit Gameboy, Neon Magenta, Warm Vintage, and Noir Monochrome.
*   **Stage Lighting:** Adjust the ambient environment brightness or add a Blush Sunset tint.
*   **Custom Templates:** Cycle through default pixel frames or upload your own transparent `.png` layout!
*   **Cross-Platform:** Tailored to work flawlessly on both Desktop PCs and Smartphones.

How to Use (Mobile & PC)

1. Open the live GitHub Pages link on your device.
2. Grant the browser **Camera Permission** when prompted.
3. Select your favorite **Pixel Frame**, **FX Filter**, and **Stage Lighting**.
4. Smile and hit **CAPTURE PHOTO**!

Mobile Users Note
*   **iOS/Safari:** When you capture a photo, if it opens the image in a new tab instead of downloading automatically, simply **long-press the photo** and tap **"Add to Photos"**.
*   **Android/Chrome:** Photos will save directly into your system's `Downloads` folder or gallery album.

 Built With
*   HTML5 Canvas & MediaDevices API (Camera stream access)
*   CSS Grid & Flexbox (Responsive pixel UI)
*   Google Fonts (*Press Start 2P*)
